NB : le composant qc-external-link est ouvert, c-à-d. qu'il ne contient pas de DOM d’ombre (shadow DOM).

Haut de page

Le composant haut de page est la flèche qui permet à l’utilisateur de remonter vers le haut de page. Il s'affiche quand l’utilisateur commence à défiler vers le haut. Il disparaît quand l’utilisateur commence à défiler vers le bas.

Documentation technique

Documentation technique

Attributs de qc-radio-group
Nom Valeur attendue Valeur par défaut Description
name Texte Nom de l’ensemble des boutons radios du groupe
legend Texte Libellé du groupe
compact na na Affiche les boutons radios au format compact
required na na Indique aux internaute que la sélection est obligatoire en ajoutant un astérisque dans la légende
disabled na na Désactive le groupe
invalid na na Indique que le groupe est dans un état invalide, et affiche le message d’erreur. (Cette valeur est à ajuster lors de la validation du formulaire, cf « Exemple de validation de formulaire avec des boutons radio ».)
NB : `invalid` prend automatiquement la valeur `false` dès qu’un bouton est coché.
invalid-text Texte Champ obligatoire/Required field Texte qui s'affiche sous les boutons radio en cas d’état invalide
Attributs de qc-radio-button

Ce composant ne peut être utilisé en dehors d’un groupe (`qc-radio-group`)

Nom Valeur attendue Valeur par défaut Description
value Texte Valeur associée au bouton radio
label Texte Libellé du bouton radio
checked na na Indique que le bouton est sélectionné par défaut.
disabled na na Désactive le bouton radio
radio-* Tout attribut commençant par radio- sera ajouté comme attribut au bouton radio brut.

Classes CSS

Nom Description
qc-textfield-container Conteneur principal du champ de texte, incluant le libellé, le champ et les messages d’erreur.
qc-textfield Boîte englobante du champ de saisie (input ou textarea), stylisée selon l’état et la taille.
qc-textfield--sm, --md, --lg, --xl, --xxl Classes de taille qui définissent la largeur maximale du champ.
qc-textfield-row Disposition horizontale de plusieurs champs de texte côte à côte.
qc-textfield-description Texte descriptif affiché sous le label pour donner des instructions ou du contexte.
qc-textfield-required Indicateur visuel (astérisque rouge) pour les champs obligatoires.
qc-textfield-charcount Affiche le nombre de caractères restants ou en trop, selon la limite définie.
max-reached Appliqué au compteur de caractères lorsque la limite est dépassée.
error Appliqué au champ lorsqu’il est invalide (vide ou trop long).
qc-form-error Conteneur du message d’erreur affiché sous le champ, incluant l’icône et le texte.
qc-icon Icône utilisée dans le message d’erreur, stylisée selon le type.
qc-disabled Appliqué au conteneur pour désactiver le champ et ses éléments.
