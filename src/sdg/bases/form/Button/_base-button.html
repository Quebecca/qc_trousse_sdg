Exemples

Classes CSS

Classe / Attribut Description
qc-button Classe de base pour tous les boutons. Définit la typographie, la taille, les marges, le padding, et le comportement responsive.
qc-compact Variante compacte du bouton avec des dimensions réduites et un padding plus petit.
qc-button-rounded Ajoute un rayon de bordure pour arrondir les coins du bouton.
qc-primary Bouton principal avec fond bleu et texte blanc.
qc-secondary Bouton secondaire avec fond blanc et texte bleu.
qc-tertiary Bouton tertiaire avec fond blanc et texte bleu.
qc-danger Bouton d’avertissement avec fond rouge et texte blanc.
qc-simple Bouton sans fond ni bordure.
disabled État désactivé du bouton. Change l’apparence et désactive les interactions.
qc-icon Classe de base pour les icônes intégrées dans les boutons. Utilise mask-image pour afficher une icône SVG.
qc-custom-icon Classe à ajouter à qc-icon pour afficher une image personnalisée via mask-image (base64 ou URL).
data-img-type Nom de l’icône à afficher.
--img-color Couleur de l’icône.
mask-image Permet d’afficher une icône personnalisée en SVG ou base64. À définir dans l’attribut style.
