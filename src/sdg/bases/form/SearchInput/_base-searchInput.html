Champ de recherche

Documentation technique

Classes CSS

Nom Description
qc-search-input Conteneur principal du champ de recherche. Il englobe l'icône, le champ de saisie et le bouton d'effacement.
qc-search-left-icon Ajoute un espacement à gauche du champ pour accueillir une icône de recherche.
qc-search-field Champ de saisie de type search, stylisé pour correspondre au design du champ de recherche.
qc-icon Icône SVG stylisée via mask ou background-image, utilisée pour la loupe et le bouton d’effacement.
qc-icon-button Bouton contenant une icône, utilisé pour effacer le champ de recherche.
qc-disabled Classe appliquée au conteneur lorsque le champ est désactivé.
