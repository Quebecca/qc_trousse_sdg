Attributs
Nom Valeur attendue Valeur par défaut Description
id Texte Chaîne de caractères aléatoire Identifiant de la liste déroulante
label Contenu HTML "" Libellé de la liste déroulante.
N.B. : Le contenu de l'élément label en slot est utilisé en priorité.
width "xs", "sm", "md", "lg", "xl" "md" Largeur de la liste déroulante. Voir l'illustration des tailles de champ texte pour une indication visuelle des tailles disponibles.
enable-search na na Affiche un champ de recherche instantanée dans le haut des options de la liste déroulante
required na na Indique aux internautes que la sélection est obligatoire en ajoutant un astérisque dans le libellé
disabled na na Désactive la liste déroulante
N.B. : L'attribut disabled de l'élément select en slot est utilisé en priorité.
invalid na na Indique si la liste déroulante est dans un état invalide
NB : invalid prend automatiquement la valeur false dès que l'attribut value est modifié.
invalid-text Texte Le champ (libellé du champ) est obligatoire. / (libellé du champ) field is required. Message d’erreur à afficher lorsque la liste déroulante est invalide
placeholder Texte Faire une sélection / Select an option Placeholder de la boîte de la liste déroulante
N.B. : Si non précisé, le placeholder prend le texte de la première option avec une valeur vide (value=""). Voir l'exemple de liste déroulante avec choix unique.
search-placeholder Texte "" Placeholder du champ de recherche
no-options-message Contenu HTML Aucun élément/No item Texte indiquant l'absence d'options dans la liste.
multiple na na Active la sélection multiple.
N.B. : L'attribut multiple de l'élément select est utilisé en priorité.
