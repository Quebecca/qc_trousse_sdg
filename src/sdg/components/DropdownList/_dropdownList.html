Liste déroulante (qc-select)

Exemples

Attributs de qc-dropdown-list

Nom Valeur attendue Valeur par défaut Description
label Contenu HTML "" Libellé de la liste déroulante
NB : l'autre possibilité est d'ajouter un élément <label> dans le composant.
width "xs", "sm", "md", "lg", "xl" "md" Largeur de la liste déroulante
value Texte Valeur(s) sélectionnée(s) dans la liste déroulante
enable-search na na Affiche un champ de recherche dans le haut des options de la liste déroulante
required na na Marque le champ comme obligatoire en ajoutant un astérisque dans le libellé
disabled na na Désactive la liste déroulante
invalid na na Affiche le message d'erreur.
NB : invalid prend automatiquement la valeur false dès qu'une donnée est sélectionnée..
invalid-text Texte Le champ `label` est obligatoire. Message d’erreur à afficher lorsque la liste déroulante est invalide
placeholder Texte "" Placeholder de la boîte de la liste déroulante
search-placeholder Texte "" Placeholder réservé du champ de recherche
no-options-message Contenu HTML Aucun élément/No item Texte en cas d'absence d'options dans la liste
multiple na na Active la sélection multiple
