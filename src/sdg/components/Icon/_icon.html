Icônes (qc-icon)

Le composant qc-icon permet d’afficher une icône avec la bonne couleur quel que soit le thème.

Exemples

Documentation technique

Attributs

Nom Valeur attendue Valeur par défaut Description
type Texte "" Nom de l’icône à afficher, à prendre dans la liste ci-dessous.
label Texte "" Texte alternatif pour l’accessibilité. Sert de libellé pour les lecteurs d’écran.
rotate Nombre na Rotation de l'image en degrés dans le sens horaire.
size "xs", "sm", "md", "lg" ou "xl" "md" Taille de l’icône.
color Toute valeur correspondant à un jeton de couleur une fois précédée de ‑‑qc‑color‑. Par exemple, "text-primary" donne la couleur de --qc-color-text-primary text-primary Couleur de l'icône.
width Texte "auto" Largeur personnalisée de l’icône.
height Texte "auto" Hauteur personnalisée de l’icône.
src URL "" Chemin vers une image personnalisée à utiliser comme icône.
