Documentation technique

Attributs

Nom Valeur attendue Valeur par défaut Description
label Texte "" Texte du libellé du champ
size "xs", "sm", "md", "lg", "xl" lg pour les champs simple (input)
xl pour les champs multilignes (textarea), 		Largeur du champ
required na na Marque le champ comme obligatoire
description Texte "" Texte descriptif affiché sous l'étiquette
max-length Nombre null Indique la limite de caractères pour le champ
invalid na na Indique que le champ est invalide en affichant un message selon l'erreur.
Français
Le champ XXX est obligatoire.
La limite de caractères du champ XXX est dépassée.
Anglais
XXX field is required.
The character limit for the ${label} field has been exceeded.
invalid-text Texte aucune Message d'erreur personnalisé affiché lorsque le champ est invalide
invalid-on-blur na na Active la validation de la case à cocher sur un changement de focus

Propriété js

Nom Valeur Description
invalidAtSubmit Booléen Indique si le champ serait invalide s'il était soumis. La vérification porte sur required et max-length
maxlengthReached Booléen Indique si la limite de caractère est dépassée.
