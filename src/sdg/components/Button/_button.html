Attributs de qc-button

Nom Valeur attendue Valeur par défaut Description
variant Texte "primary" Définit le style visuel du bouton. Valeurs possibles : primary, secondary, tertiary, danger.
label Texte "" Texte affiché dans le bouton.
compact "True" ou "False" False Affiche le bouton dans un format plus petit.
rounded "True" ou "False" False Ajoute un rayon aux coins du bouton pour un style arrondi.
disabled "True" ou "False" False Désactive le bouton et empêche toute interaction.
icon Texte "" Nom d’une icône présente dans la librairie du projet. (voir la section icônes)
icon-position "left" ou "right" "left" Positionne l’icône à gauche ou à droite du texte dans le bouton.
icon-src Tout format accepté par la fonction css url() "" Permet d’afficher une image personnalisée.
