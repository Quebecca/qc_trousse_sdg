Attributs

Nom Valeur attendue Valeur par défaut Description
type "general" ou "warning" "general" Type de l’alerte : s'il s'agit d’une alerte bleue ou jaune
maskable "true" ou "false" "true" Afficher le bouton de fermeture de l’alerte
content Texte Chaine vide Afficher le contenu de l’alerte
hide "true" ou "false" "false" Masque ou affiche l’alerte
full-width "true" ou "false" "false" Affiche le contenu de l’alerte dans un conteneur pleine-largeur (classe css qc-container-fluid)

Événement JS

Un événement JS qc.alert.hide est émis à la fermeture de l’alerte
