Ajout dans un libellé de champ texte

Documentation technique

Attributs

Nom Valeur attendue Valeur par défaut Description
text Texte Chaine vide Texte lié à l'infobulle
description Texte Chaine vide Contenu de l'infobulle
icon "information" ou "question" "information" Icône de l'infobulle
display-mode "popover" ou "modal" "popover" Affichage phylactère (popover) ou feuille (modal). Dans tous les cas, en mobile l'affichage est en feuille.
position "right", "top" ou "bottom" "top" Position du phylactère
description-id Id d'un élément dans le document Nulle Id d'un élément dans la page, dont le contenu sera affiché dans l'infobulle.
